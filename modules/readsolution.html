ReadSolution

Reads all the source files from a specified msbuild solution. Note that this requires the MSBuild tools to be installed (included with Visual Studio).
The output of this module is similar to executing the ReadFiles module on all source files in the solution.

Usage

  • ReadSolution(DocumentConfig path)

    Reads the solution file at the specified path. This allows you to specify a different solution file depending on the input.

    • path

      A delegate that returns a string with the solution file path.

  • ReadSolution(string path)

    Reads the solution file at the specified path.

    • path

      The solution file path.

Fluent Methods

Chain these methods together after the constructor to modify behavior.

  • WhereFile(Func<string, bool> predicate)

    Filters the source code file based on path.

    • predicate

      A predicate that should return true if the source code file should be included.

  • WhereProject(Func<string, bool> predicate)

    Filters the project based on name.

    • predicate

      A predicate that should return true if the project should be included.

  • WithExtensions(params string[] extensions)

    Filters the source code files based on extension.

    • extensions

      The extensions to include (if defined, any extensions not listed will be excluded).

Metadata

The following metadata is added to each document.

  • RelativeFileDir: string

    The relative directory of the file from the Wyam input folder.

  • RelativeFilePath: string

    The relative path to the file (including file name) from the Wyam input folder.

  • RelativeFilePathBase: string

    The relative path to the file (including file name) from the Wyam input folder without the file extension.

  • SourceFileBase: string

    The file name without any extension. Equivalent to Path.GetFileNameWithoutExtension(SourceFilePath).

  • SourceFileDir: string

    The full absolute directory of the file. Equivalent to Path.GetDirectoryName(SourceFilePath).

  • SourceFileExt: string

    The extension of the file. Equivalent to Path.GetExtension(SourceFilePath).

  • SourceFileName: string

    The full file name. Equivalent to Path.GetFileName(SourceFilePath).

  • SourceFilePath: string

    The full absolute path of the file (including file name).

  • SourceFilePathBase: string

    The full absolute path of the file (including file name) without the file extension.

  • SourceFileRoot: string

    The absolute root search path without any nested directories (I.e., the path that was searched, and possibly descended, for the given pattern).
