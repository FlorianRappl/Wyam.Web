Modules

Content

Append

Appends content to the document.

AutoLink

Replaces occurrences of specified strings with HTML links.

Content

Sets the content of each input document.

Prepend

Prepends content to the document.

Replace

Replaces a search string in the content of each input document with new content.

ReplaceIn

Replaces a search string in the specified content with the content of an input document.

Control

Branch

Evaluates a sequence of modules without impacting the primary module flow.

Concat

Evaluates a sequence of modules and concats the result documents with the input documents.

Documents

Replaces pipeline documents with previously processed ones

ForEach

Executes the input documents one at a time against the specified child modules.

FrontMatter

Extracts the first part of content for each document and sends it to a child module for processing.

GroupBy

Partitions a sequence of documents based on a key.

If

Provides conditional logic for your pipeline.

OrderBy

Orders the input documents based on a key.

Paginate

Partitions a sequence of documents into equally sized pages.

Where

Filters the current sequence of modules using a predicate.

Data Processing

Image

Manipulates images.

Extensibility

Execute

Executes custom code for each input document.

Trace

Outputs trace messages during execution.

Input/Output

CopyFiles

Copies files from one location on disk to another one and sets associated metadata.

ReadFiles

Reads files from disk and sets associated metadata.

WriteFiles

Writes document content to disk.

Metadata

Excerpt

Finds the first occurrence of a specified HTML element and stores it's contents as metadata.

Meta

Adds the specified metadata to each input document.

Yaml

Parses YAML content and stores the result in document metadata.

Templating

Less

Compiles Less CSS files to CSS stylesheets.

Markdown

Parses markdown content and renders it to HTML.

Razor

Parses Razor templates and renders them to HTML.
