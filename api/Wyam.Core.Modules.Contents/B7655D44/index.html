Remarks

Implements RSS and Atom feeds. All input documents should have RssTitle and RssDescription metadata because they are required by RSS. You can override these metadata keys via WithTitleMetaKey(string key) and WithDescriptionMetaKey(string key).

This module outputs input files without changes and one another file: the RSS feed. The RSS feed contains RelativeFilePath and IsRssFeed: true metadata keys. Use WriteFiles module inside Branch module without parameters to place file in appropriate file system location. See example.

Options:

  • Use WithoutGuid() to disable generation of unique identifier for each input document. Guid is generated using input document's source location. You may want to disable it if your data is generated on fly.
  • Optional RssPubDate metadata will be used to specify document's publication date. You can override this metadata key with WithPublicationDateMetaKey(string key).
  • Language can be set for feed using WithLanguage(string lang). Language string must follow these conventions: http://www.rssboard.org/rss-language-codes
  • Use WithLinkCustomizer to create pretty urls.

Some things to know:

  • If publication date for each document is not set, most RSS reader programs will assume that most recent publication is the top most in the output RSS file (that is first input document).
  • Use OrderBy module to order your documents in right order.
  • Use a feed validator when generating RSS for first time: https://validator.w3.org/feed/
  • Only basic tags are defined in this module at this moment, eg. you cannot attach author tag to specify RSS item's author.

Examples

Input/posts/hello.md: 

    ---
    RssTitle: Hello, world!
    RssDescription: My first blog post
    RssPubDate: 10/11/12
    ---
    
    Hello, RSS world!
config.wyam: 

    Pipelines.Add("Blog posts",
        ReadFiles("posts/*.md"),
        FrontMatter(Yaml()),
        Markdown(),
        WriteFiles(".html"),
        Rss(siteRoot: "http://example.org",
            outputRssFilePath: "posts/feed.rss",
            feedTitle: "My awesome blog",
            feedDescription: "Blog about something"
        ),
        WriteFiles()
    );
Output/posts/feed.rss:
