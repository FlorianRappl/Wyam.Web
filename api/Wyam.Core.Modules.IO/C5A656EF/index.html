ReadFiles Class

Wyam.Core.Modules.IO
Interfaces

Base Type

Summary

Reads the content of files from the file system into the content of new documents.

Syntax

public class ReadFiles : IModule, IAsNewDocuments

Constructors

Name Summary
ReadFiles(DocumentConfig) Reads all files that match the specified path. This allows you to specify different search paths depending on the input.
ReadFiles(string) Reads all files that match the specified search pattern.

Methods

Name Value Summary
Execute(IReadOnlyList<IDocument>, IExecutionContext) IEnumerable<IDocument>
FromAllDirectories() ReadFiles Specifies that all directories should be searched.
FromTopDirectoryOnly() ReadFiles Specifies that only the top-level directory should be searched.
Where(Func<string, bool>) ReadFiles Specifies a predicate that must be satisfied for the file to be copied. The input to the predicate is the full path to the source file.
WithExtensions(string[]) ReadFiles Specifies that only files with the given extensions should be read.
WithSearchOption(SearchOption) ReadFiles Specifies whether to search all directories or just the top directory.

Remarks

For each output document, several metadata values are set with information about the file. This module will be executed once and input documents will be ignored if a search path is specified. Otherwise, if a delegate is specified the module will be executed once per input document and the resulting output documents will be aggregated. In either case, the input documents will not be returned as output of this module. If you want to add additional files to a current pipeline, you should enclose your ReadFiles modules with Concat.
